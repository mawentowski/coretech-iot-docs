PrintBill

Message with the data to print a payment bill, the print data is standard format for a payment terminal printer.

Message Properties

Field Type Mandatory/Optional Value/Description
MessageNamespace string M payment
MessageType string M PrintBill
MessageVersion string M 1
AttachmentId string M {attachmentId}
CorrelationId string M {user_defined_string}
DeviceId string M {deviceId}
Timestamp DateTime M {timestamp}

Message Body

Base Message Body

Field Type Description/Values
MerchantBill Bill Holds the bill data for the merchant
CardholderBill Bill Holds the bill data for the customer

Bill

Field Type Description/Values
BillHeader BillElement[] The header section of the bill
BillContent BillElement[] The main content of the bill
BillFooter BillElement[] The footer section of the bill
BillElement
Field Type Description/Values
Format string Print format
Name string The field name to print
Abbreviation string The abbreviated field name
Value string The value text to print
NameRequired string true
