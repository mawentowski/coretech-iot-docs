Set up Service Bus Console

Use the SB Console to view Service Bus messages

The Service Bus Console allows you to view Service Bus responses. The Console takes the linkage details returned after a Link call and attachs to that linkage. Then, when you attach to and interact with devices, messages related to those events are displayed in the console.

In reality, a customer would use the IoT Message Service SDK provided by Microsoft to receive messages on the Service Bus. CoreTech staff use the console to simulate how a live customer would receive messages without the need of creating an application.

Install the Console

Service Bus
  1. Install the following programs, if not already installed:
    • Z-zip - to extract the console file package.
    • Notepad++ - to open the .json file
    • .NET 5.0 Runtime - required for the console to run.
  2. In File Explorer, right-click ServiceBusConsole.7z file, hover over 7-zip and select Extract files...
  3. Leave the default settings as is and click Okay.
    • A new folder displays containing the extracted files.

Configure the Console

Service Bus
  1. Open the new ServiceBusConsole folder.
  2. Right-click AttachmentDetails.json and select Edit with Notepad++.
  3. In Postman, send a sync/v2/linkages request.
  4. Copy the JSON returned in the Response Body, and paste it into the AttachmentDetails.json, replacing its current contents.
  5. Save and close the file.

Run the Console

Service Bus
  1. Back in the extracted folder, double-click the CoreTech.IoTApi.ServiceBusConsole executable to run the Console.
    • Look for the file named CoreTech.IoTApi.ServiceBusConsole whose type is Application (as seen in the Type column in Windows Explorer).
    • The console displays.
  2. Hit Enter and the console will link to the service bus linkage entered earlier.
    • Device messages returned on the Service Bus are now displayed in the console in real-time.
    If you hit Enter twice, you have aborted the attachment to the Service Bus and will not receive messages. To resolve, close the console, run it, then hit Enter (one time).
