Start Payment Process

Initiate a payment process on a payment device

Send a POST /remit/v2starttransaction request to the payment API to start a payment process.

Make an API Call

For information on the common components that make up a request, like the Resource URL, Header Parameters, or the Authorization Header, see API Call Components.

"+`
}`)

    client := &http.Client {
    }
    req, err := http.NewRequest(method, url, payload)

    if err != nil {
    fmt.Println(err)

    }
    req.Header.Add("Content-Type", "application/json")

    fmt.Println(err)

    return
    }
    fmt.Println(string(body))
}

Sample POST /remit/v2starttransaction request to the payment API to start a payment process.

Request Body

The following is a sample Request Body for using a PMC device.

A relationId must be passed in order for the Remit API to uniquely identify the transaction.

Request Body
                
{
    "attachmentId": "{{pmc_attachment_id}}",
    "relationId": "correl_paymt",
    "Amount": 127.30,
    "CurrencyCode": 978,
    "OriginatorsTransactionReference":"{{$guid}}"
}

POST /remit/v2starttransaction request

Response

When a POST /remit/v2starttransaction request is sent, a payment transaction is initiated and a ProcessPaymentResponse message is displayed on the Service Bus.
